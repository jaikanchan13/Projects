# Groq AI Chatbot

A simple command-line AI chatbot built with Python and the Groq API.

## Features
- Uses Groq's Llama model
- Stores conversation history during a session
- Loads API key securely from a `.env` file
- Runs in the terminal

## Tech Stack
- Python
- Groq API
- python-dotenv
